📦 go.xiexianbin.cn Go 语言开源库

🔧 常用工具库

golib

通用工具函数集合，包含：

  • 命令行执行
  • logger
  • 安全的并发map

安装方式：

go get github.com/xiexianbin/golib

🚀 Gorm Pagination plugin

gorm-paginate

GORM 分页插件：

  • 分页
  • 搜索
  • 排序

快速开始：

import "go.xiexianbin.cn/gorm-paginate"

🚀 xgorm

xgorm

基于Go 1.18+的泛型特性，实现Gorm框架通用的CRUD（增删改查）操作方法

快速开始：

import "go.xiexianbin.cn/xgorm"

🚀 authx

authx

oauth2 server

快速开始：

import "go.xiexianbin.cn/authx"

🚀 authkit

authkit

一个支持多种第三方 OAuth（如 GitHub, Google）登录和获取用户信息的 golang 实现

快速开始：

import "go.xiexianbin.cn/authkit"
